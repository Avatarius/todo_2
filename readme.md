## Установка и запуск
Для установки и запуска проекта необходимо выполнить команды

```
npm install
npm run start
```

## Запуск jest тестов

Для установки и запуска проекта необходимо выполнить команды

```
npm run test
```

## Запуск cypress тестов

Для установки и запуска проекта необходимо выполнить команды

```
npm run cypress:open
```
